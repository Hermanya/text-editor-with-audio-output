For the best experience: go offline on chrome for OS X

If you are online, chrome uses google for speach synthesis. Not only their default settings (voice, language) are weird, but it also causes a noticable delay.

How to use

Insert mode (on by default)
escape
activate normal mode
space
space and say last word
delete
delete last word
other characters
should work as expected
Normal mode
i
activate insert mode
l
say entire text
delete
works in this mode too
r
evaluates as javascript

Tips

  • Reload the page to start from scratch
  • text is a global variable
  • Script itself is <100 lines
  • Mappings for silent characters
